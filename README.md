# Laundry Services Web Application

A lightweight single-page web application built for a localized laundry business. This project allows users to view available services, dynamically add or remove items from an invoice cart panel, see live total calculations, and submit bookings directly via frontend email integration.

## 🚀 Live Link
View the live project here: **[laundryservicekanpur.netlify.app]**

---

## 💻 Features
* **Dynamic Cart Management:** Pure JavaScript arrays handling live adding and removing of items without reloading the page.
* **Instant Total Calculation:** Automatic math loops recalculating prices seamlessly as items change.
* **Serverless Email Booking:** Connected directly to EmailJS to forward customer details and ordered services straight to a mailbox.
* **Responsive Grid Layout:** Built using modern utilities grid framework to align contents side-by-side cleanly.

---

## 🛠️ Built With
* **HTML5:** Basic page setup structures.
* **Tailwind CSS:** Minimal style configurations and grid alignments.
* **Vanilla JavaScript:** Event hooks, loops, and DOM text injections.
* **EmailJS SDK:** External connection client to securely handle email dispatching.

---

## 📁 Project Files
* `index.html` - Contains the layouts, hero text, forms, and tables.
* `script.js` - Contains the data list array, pricing logic, and EmailJS connection configurations.
